# Maricela MK Landing

Landing page React para Maricela Monchez, basada en la version premium `space-landing.html`.

## Uso

```bash
npm install
npm run dev
```

## Estructura

- `index.html`: entrada principal del sitio.
- `src/App.jsx`: experiencia React completa.
- `src/styles.css`: estilos visuales y animaciones.
- `public/assets`: imagenes usadas por la landing.
- `space-landing.html`: referencia original exportada.
